# 坑：LOGID 生成问题

```
RPCS=8090
LOGID=$(\
  go run github.com/google/trillian/cmd/createtree \
  --admin_server=:${RPCS} \
) && echo ${LOGID}
```

应修改为：

```
RPCS=8090
LOGID=$(\
  go run github.com/google/trillian/cmd/createtree \
  --admin_server=127.0.0.1:${RPCS} \
) && echo ${LOGID}
```

# 坑：demo 启动问题
```
go run github.com/DazWilkin/simple-trillian-log-1 \
--tlog_endpoint=:8090 \
--tlog_id=${LOGID}
```

应修改为：

```
go run github.com/DazWilkin/simple-trillian-log-1 \
--tlog_endpoint=127.0.0.1:8090 \
--tlog_id=${LOGID}
```

此外，demo 内有一处简单问题，根据报错提示简单修改下即可。
